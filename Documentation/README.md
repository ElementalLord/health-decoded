# Health Decoded Documentation

This directory contains the project’s governing documents and technical decisions.

- [Constitutions](./Constitutions/) define enduring engineering and security rules.
- [Project Playbook specifications](./Project%20Playbook%20SPECS/) define product and implementation requirements.
- [Architecture Decisions](./Architecture%20Decisions/) record approved decisions that resolve conflicts or set prototype scope.
- [Curriculum](./Curriculum/) contains cross-lesson audits, medical-review notes, and optimization decisions.

When documents conflict, follow the authority order recorded in the architecture decision record.
